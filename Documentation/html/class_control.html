Detailed Description

Input and control flow managment.

Control class is created to manage chain of characters entered by the user. The class object take parameters from the input stream, dvivide them on commands(all commands have flag name and arguments) and gather wrong form of input in incorrect flags container.

Static class methods are used to execute features from this software.
